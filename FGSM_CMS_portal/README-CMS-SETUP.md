# ФГСМ портал со административен панел

Овој пакет содржи статичен ФГСМ портал со Git-backed CMS. Уредниците додаваат содржина преку `/admin/`, а веб-страницата автоматски ја прикажува во професионалниот дизајн.

## Што може да се уредува

- Почетна секција, истакната активност, цели и контакт
- Новости со фотографии и целосни статии
- Календар и натпревари по гранка
- Резултати со PDF датотеки
- Документи, правилници, програми и обрасци
- Раководство, тела, национални тимови, тренери и судии
- Галерија
- Партнери и спонзори
- Социјални објави/линкови
- Содржина за секоја внатрешна route страница

## Поставување на Netlify

1. Ставете ја целата папка во GitHub repository. Главната гранка треба да биде `main`.
2. Во Netlify изберете **Add new site → Import an existing project** и поврзете го repository-то.
3. Build command оставете празно. Publish directory е `.`.
4. Во Netlify отворете **Identity** и активирајте го сервисот.
5. Во **Identity → Services** активирајте **Git Gateway**.
6. Во **Identity → Registration preferences** поставете **Invite only**.
7. Поканете ги уредниците преку нивните е-пошта адреси.
8. Отворете `https://ВАШ-ДОМЕН/admin/` и најавете се.

## Работен процес

- **Save** ја зачувува објавата како нацрт.
- **Set status → In review** ја праќа на проверка.
- **Publish** ја објавува содржината.
- Секоја промена е верзионирана во Git и може да се врати.

## Важно

- Не бришете ги `content/*.json` датотеките.
- Сликите и PDF датотеките автоматски се чуваат во `assets/uploads/`.
- За внатрешните страници користете точен route ID од `ROUTE-CATALOG.md`.
- Веб-страницата мора да се отвора преку web server/Netlify. При директно отворање со `file://`, браузерот може да блокира читање на JSON датотеките; вградениот fallback сепак ќе го задржи изгледот.
